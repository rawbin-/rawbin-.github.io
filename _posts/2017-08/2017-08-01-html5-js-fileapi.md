#### 十六进制内容查看

- Sublime Text 安装 HexViewer插件

- Visual Studio Code 安装 HexDump插件

- VIM + xxd命令查看十六进制

  ```
  vim -b FILENAME
  #进入vim 界面后
  :%!xxd.  # %代表当前文件, !表示后面的命令执行 xxd转换二进制为16进制
  ```
